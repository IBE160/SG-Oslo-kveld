# Project Plan – To Like (IBE160)

Dette dokumentet beskriver den overordnede planen for utviklingen av «To Like»,
et KI-assistert memory-spill laget i emnet IBE160 Programmering med KI.

## 1. Prosjektoversikt
Målet med prosjektet er å utvikle et enkelt, spillbart memory-spill der brukeren kan:
- velge antall spillere (2+)
- velge antall kort (partall)
- velge modus (tall eller bokstaver)

Prosjektet er utviklet med støtte fra KI-agenter (Analyst, PM, Designer, Developer, Tester)
i tråd med BMAD-rammeverket.

## 2. Viktige leveranser
- Fungerende applikasjon (Next.js + TypeScript + Zustand)
- BMAD-dokumentasjon (plan, PRD, arkitektur, epics, workflow-status)
- Implementert spill med grafikk, layout og brukeropplevelse
- Refleksjonsrapport (egen fil – ikke i repo)
- Prompter brukt i prosessen (mappe: /prompts)

## 3. Planleggingsfaser (BMAD)
BMAD deler prosessen inn i:

### Fase 1 – Analyse og forståelse
- Definere problemet
- Lage prosjektbeskrivelse
- Lage PRD og user flows

### Fase 2 – Design og arkitektur
- Lage UX-skisser og visuell retning
- Lage arkitektur for teknisk løsning
- Lage epics og user stories basert på arkitekturen

### Fase 3 – Utvikling
- Implementere spilllogikk
- Lage UI
- Testing og feilretting

### Fase 4 – Publisering
- Klargjøre README
- Kode i main-branchen
- Dokumentasjon ferdig i /docs/

## 4. Nåværende status
Prosjektet er i sluttfasen:
- Koden er funksjonell
- Dokumentasjonsfilene i /docs/ er opprettet
- Refleksjonsrapport ferdigstilles
- Prompts legges inn
