# Rollerfordeling «To Like»

## Bidrag i prosjektet «To Like»

### Nihad
- Har hatt hovedansvar for koden i spillet.
- Satt opp logikken i Next.js, React og Zustand.
- Koblet sammen sidene, komponentene og spillflyten.
- Samarbeidet med KI-verktøy for å generere og justere kode.

### Mehmet
- Har hatt hovedansvar for grafikk og utseende.
- Jobbet med farger, bakgrunn, layout og visuelle effekter.
- Justert utseendet på kortene og brettet slik at spillet ser ryddig og spillvennlig ut.
- Testet spillet, oppdaget feil og foreslått forbedringer.
- Skrevet README og vært pådriver for dokumentasjon og struktur i repoet.
